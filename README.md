# rn-location-hook

Hook that use expo-location to retrieve device coordinates

## Installation

```sh
npm install rn-location-hook
```

## Usage

```js
import RnLocationHook from "rn-location-hook";

// ...

const result = await RnLocationHook.multiply(3, 7);
```
